Weka API Key Yako

UNIPATH inahitaji API Key ya Google Gemini ili iweze kutoa majibu kutoka mtandaoni. Ni bure kupata!

Hatua za kupata API Key:
1️⃣ Nenda aistudio.google.com/app/apikey
2️⃣ Ingia na akaunti yako ya Google
3️⃣ Bonyeza Create API Key
4️⃣ Copy key (inaanza na AIza...)
5️⃣ Paste hapa chini 👇

⚠️ Key si sahihi – angalia tena

🔒 Key yako inahifadhiwa kwenye simu yako tu – haipelekwi mahali pengine.
Kila mwanafunzi aweke key yake mwenyewe bure.
